Keys & Configuration

All sensitive keys are stored as Netlify environment variables — never in this HTML file. Follow the steps below to configure them.

This is the right way to store secrets. Environment variables in Netlify are encrypted, never exposed to the browser, and only accessible by the server-side proxy functions that run on Netlify's infrastructure.
Required environment variables
Variable nameWhat it holdsExample value
ANTHROPIC_API_KEY Your Anthropic API key. Used by the Claude proxy to parse uploaded documents. Get it from console.anthropic.com › API Keys. sk-ant-api03-...
GOOGLE_SCRIPT_URL Your Google Apps Script Web App URL. Used to read and write SOPs to your Google Sheet database. Set up per the Setup Guide. https://script.google.com/macros/s/.../exec
SITE_PASSWORD The shared password for the login screen. All managers use this same password. Change it anytime by updating this variable in Netlify. natals2026 (change this)
How to set environment variables in Netlify
  1. Log in to app.netlify.com and open your site
  2. Click Site configuration › Environment variables
  3. Click Add a variable
  4. Enter the variable name exactly as shown above (all caps, underscores)
  5. Paste the value and click Save
  6. Repeat for all three variables
  7. Go to Deploys and click Trigger deploy so the new variables take effect
You must trigger a new deploy after adding or changing environment variables. Netlify only reads them at deploy time for serverless functions.
